# Data & ML Workflow Skill

## Intent
Streamline the development of data-centric features and machine learning models.

## When to Use
- Designing data schemas or intake pipelines.
- Implementing feature engineering or model training scripts.
- Visualizing or analyzing dataset performance.

## Workflow
1. **Data Discovery**: Understand the source, schema, and quality of incoming data.
2. **Preprocessing**: Clean, transform, and version datasets.
3. **Modeling**: Implement training loops, handle hyperparameters, and evaluate results.
4. **Deployment**: Package models for inference and monitor performance.
