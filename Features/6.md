# Migration Wizard – Overview

## Goal

Help developers migrate existing AI context files into Rosetta’s `.ai/` architecture with minimal manual work.

## Supported Sources (v1)

- `CLAUDE.md`-only repos.
- `.cursorrules`-only repos.
- `.github/copilot-instructions.md`-centric repos.
- Mixed setups with partial overlap.

## High-Level Flows

- `rosetta migrate` (interactive wizard):
  - Detect existing files.
  - Ask how to construct `.ai/master-skill.md`.
  - Offer to generate `.ai/AGENT.md`, `.ai/task.md`, memory/logs.
  - Regenerate IDE wrappers to point to `.ai/`.

- Specialized commands (optional later):
  - `rosetta migrate-from-claude`
  - `rosetta migrate-from-cursor`

## Outputs

- `.ai/master-skill.md` containing merged or selected context.
- `.ai/AGENT.md`, `.ai/task.md`, `.ai/memory/*`, `.ai/logs/*`.
- Updated wrappers that reference `.ai/`.
- Backups of original files (`*.bak`) to prevent data loss.
