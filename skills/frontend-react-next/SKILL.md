# Frontend: React/Next.js Skill

## Intent
Expertly implement user interfaces, state management, and routing using React and Next.js.

## When to Use
- Building new UI components or pages.
- Managing client-side or server-side state.
- Optimizing performance (LCP, FID, CLS).

## Workflow
1. **Component Design**: Break down UI into modular, reusable components.
2. **State Management**: Use appropriate hooks (useState, useMemo) or global state.
3. **API Integration**: Fetch data using server components or client-side hooks.
4. **Testing**: Add component or snapshot tests to prevent UI regressions.

## Guidelines
- Follow patterns for None.
- Respect accessibility and performance constraints in None.
- Maintain consistency with the design system.
